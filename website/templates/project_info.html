Information about this project

This is a schoolproject for the Hanze University of applied sciences in which I will be making this website. On this website there will be the FastQC-tool available. I will be using HTML, CSS, Python, Flask en Jinja to make this happen. At the bottom of the page you can click the github logo to visit the repository for this project. The image shows the planning for this project.

The project is split into 7 phases.

  1. Writing the project proposal
  2. Creating the static website
  3. Making the static website dynamic
  4. Creating the results page
  5. Making the OOP backend
  6. Linking backend and results page
  7. Finishing the website and making and doing the final presentation
